# 关于
本文档由复旦大学数学科学学院的志愿者群体编写，目前处于搭建阶段。

本手册的目的是将数院学子本科发展的经验加以总结，为学弟学妹提供生涯发展上的参考，减少信息差带来的不公平，降低出国（境）留学、国内升学以及就业的准备成本。

正所谓 *数苑尝百味，人生大胸怀[^1]*：这本手册无意于给你细致入微的指导，只希望读者能在前辈故事的感召下，品读只属于自己的四载春秋。
